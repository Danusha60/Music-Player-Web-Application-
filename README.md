🎧 MelodyHub – Music Player Web Application

MelodyHub is a responsive and interactive Music Player Web App designed to provide users with a smooth and engaging music listening experience. Built using pure HTML, CSS, and JavaScript, the application focuses on clean UI design, seamless song controls, and responsive performance across devices.
🚀 Features
🎵 Interactive song library with card-based layout
🌐 Language-based filtering (English, Telugu, Hindi)
📻 Fixed music player with album art, song title, and artist details
⏯ Music controls: Play, Pause, Next, Previous
🔁 Automatically plays the next song when the current one ends
📱 Fully responsive design for desktop and mobile devices

🛠️ Tech Stack
HTML – Structure and layout
CSS– Styling and responsive design
JavaScript – Functionality and interactivity

💡 Project Objective
The goal of MelodyHub is to create a user-friendly and visually appealing music player application using core web technologies without relying on external frameworks. This project helped us improve our frontend development, DOM manipulation, event handling, and responsive design skills.

👨‍💻 Team Collaboration
This project was developed as part of a hackathon, where teamwork, creativity, and problem-solving played a major role in building the final product successfully.
